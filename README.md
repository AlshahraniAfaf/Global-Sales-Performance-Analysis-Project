# Global Sales Performance Analysis

## 📌 Project Overview
An interactive Power BI dashboard focused on analyzing global sales distribution and order segmentation. This project demonstrates the ability to transform raw transactional data into actionable insights with a high-end, professional user interface.

## ❓ Business Questions
- What is the total revenue, quantity sold, and customer reach?
- How do sales trend over a complete date hierarchy (Revenue Over Time)?
- Which are the top 5 best-selling products?
- What is the proportion of Large vs. Small orders?

## 🛠 Tech Stack
- 📊 Power BI Desktop – Report creation and interactive analytics.
- 🧠 DAX & Power Query – Used for data cleaning and creating business logic.
- 📝 Data Modeling – Established relationships and date hierarchies for deep-dive analysis.

## 🚀 Features & Logic

### 1. Custom Calculated Column (Order Category)
The core of the segmentation analysis involved creating a calculated column to classify orders based on their volume:
- Small Orders: Standard individual transactions.
- Large Orders: Bulk or high-value transactions.
> *This classification is visualized in the "Sales by Order Size" Pie chart.*

### 2. Key Performance Indicators (KPIs)
Direct visibility into four main metrics:
- Total Revenue: $8.9M
- Total Quantity: 5M
- Total Customers: 4K
- Avg Order Value: 481

### 3. Visual Components
- Home Page: A professional entry screen with project objectives and overview.
- Revenue Over Time: Line chart showing trends across Years, Quarters, and Months using a Date Hierarchy.
- Global Sales Map: Spatial distribution of sales volume across different countries.
- Top 5 Products: A bar chart identifying the highest-grossing items.

## 💡 Key Insights
- Order Breakdown: 79.6% of orders are classified as Small, while 20.4% are Large.
- Revenue Trend: The "Revenue Over Time" analysis shows a consistent growth pattern over the recorded timeline.
- Geographic Reach: Clear identification of top-performing regions through the interactive map.
